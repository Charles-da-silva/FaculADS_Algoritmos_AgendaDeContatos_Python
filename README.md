# Trabalho de faculdade - Agenda de contatos em Python

Trabalho da faculdade, na cadeira de Algoritmos de programação, onde criamos uma agendad e contatos em Python, utilizando while, try/except, match, for in, manipulação de listas/arrays, escopo de variáveis, dentre outros.

O exercício pedia que fosse criada uma agenda de contatos, com ID para cada contato, que fosse possível cadastrar novos, listas todos, listar por ID ou ocupação e excluir.
